# Inforce-test-task

## Start web-server
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Start json-server

If you don't have json-server on your machine, just install it
```bash
  npm i -g json-server
```
Start the server
```bash
  json-server -p 4000 db.json
```
